Population & Armée

Actions

1) Cliquer sur un point que vous possédez
2) Cliquer sur un autre point pour construire une route
3) Sélectionner une route pour envoyer des soldats

Notifications

Comment jouer (V0)

  • Sélectionner une carte (Europe, Amériques, Asie).
  • Placez vos zones de ressources en cliquant sur le bouton « Placer ressource », puis sur un point ressource vous appartenant.
  • Construisez des routes entre vos points : clic sur le point A puis le point B.
  • Les routes actives montrent un point mobile et un pourcentage d'activité.
  • Envoyez l'armée via une route sélectionnée (bouton « Envoyer armée »).
  • Attaquer : construisez une route vers un point ennemi ; l'adversaire est notifié.
  • Encerclement : si des routes ennemies entourent votre capitale, population et armée diminuent.
  • Victoire : dernier pays vivant ou contrôle majoritaire de la carte.
